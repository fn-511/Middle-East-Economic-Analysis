Middle East Economic Resilience Analysis (1990-2024)
Project Overview
This data science project explores the economic landscape of Middle Eastern countries over more than three decades. The analysis focuses on GDP growth trends, the impact of Brent oil prices on regional economies, and unemployment rates.

Key Analysis Features
Data Cleaning: Handling missing values and preparing multi-country economic indicators.

Comparative Visualization: Time-series analysis of GDP growth for multiple nations (KSA, Bahrain, Kuwait, Egypt, etc.).

Economic Insights: Visualizing the resilience of regional economies against global oil price fluctuations.

Technology Stack
Python: Core programming language.

Pandas: Data manipulation and cleaning.

Matplotlib & Seaborn: Advanced data visualization and plotting.

Jupyter Notebook: Interactive environment for analysis.

Dataset
The project utilizes a comprehensive dataset including:

GDP (Current USD & Growth %)

Brent Oil Prices

Unemployment & Life Expectancy

Foreign Direct Investment (FDI)
